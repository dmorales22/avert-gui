# AVERT GUI

This repository contains Team 3's AVERT GUI. This GUI was developed using
React and is built with Electron so that it can be used as a desktop
application.

## Installation

In order to install, you must install Node.js and Git on your machine.

After that's installed, run the following scripts:

```sh
git clone https://github.com/Team-3-CS4311/avert-gui.git

cd avert-gui

npm i
```

## Running

In order to run the project during development, run the
following:

```sh
npm run-script dev
```
